# arc-conf

A paper deadline tracker focused on **AI / Robotics / Control** conferences — including ICRA, IROS, RSS, CoRL, RA-L, TRO, NeurIPS, ICML, ICLR, L4DC, and more.

Built for researchers who want a clean, opinionated dashboard instead of a generic CS deadline list.

## Features

- Curated list of AI / Robotics / Control venues only
- Multiple deadlines per conference (abstract, paper, supplementary, rebuttal, camera-ready)
- RA-L ↔ ICRA / IROS presentation cycle tracking
- Personal submission status tracker (In Progress / Submitted / Under Review / Accepted / Rejected)
- Time zone display in AoE and your local time
- `.ics` calendar export
- Daily auto-update via GitHub Actions

## Tech Stack

- **Framework**: Next.js (App Router, Static Export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Hosting**: GitHub Pages / Cloudflare Pages
